# URLcrazy Cheat Sheet

#### Comprobar más de 2000 variantes de errores tipográficos en google.com
```
urlcrazy <Domain>
```

#### Comprobar y no resolver DNS
```
urlcrazy <Domain> -r
```

#### Comprobar y guardar los resultados
```
urlcrazy <Domain> -o <Output> -f <HumanReadable/JSON/CSV>
```
