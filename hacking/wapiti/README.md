# Wapiti Cheat Sheet

#### Escaneo de URL
```
wapiti -u <URL>
```

#### Escaneo de URL y mostrar resultados con color
```
wapiti -u <URL> --color
```

#### Escano de URL con ciertos modulos
```
wapiti -u <URL> --color -m sql,xss,xxe
```

#### Escaneo de URL y guardar un reporte en formato html
```
wapiti -u <URL> --color -f html -o report
```
