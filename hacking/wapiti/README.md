# Wapiti Cheat Sheet

#### Escaneo de URL
```
wapiti -u <URL>
```

#### Escano de URL con ciertos modulos
```
wapiti -u <URL> -m sql,xss,xxe
```

#### Escaneo de URL y guardar un reporte en formato html
```
wapiti -u <URL> -f html -o report
```
