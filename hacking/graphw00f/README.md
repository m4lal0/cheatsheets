# graphww00f Cheat Sheet

#### Detectar endpoint de GraphQL
```
./main.py -t <URL> -d
```

#### Detectar endpoint de GraphQL y ejecutar fingerprinting
```
./main.py -t <URL> -d -f
```

#### Detectar endpoint de GraphQL y ejecutar fingerprinting, guardando el resultado en un CSV
```
./main.py -t <URL> -d -f -o <OUTPUT>
```
