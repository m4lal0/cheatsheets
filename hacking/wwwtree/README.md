# wwwtree Cheat Sheet

#### Levantar webserver en el puerto 80, en un directorio e interface dado
```
wwwtree -r /opt/Windows -i eth0
```

#### Levantar un webserver en puerto diferente
```
wwwtree -r /opt/Windows -i eth0 -p <PORT>
```
