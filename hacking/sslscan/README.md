# SSLscan Cheat Sheet

#### Escaneo de servicios SSL a un Host
```
sslcan <HOST:PORT>
```

#### Escaneo de servicios SSL desde un archivo con listado de host
```
sslcan --targets=<FILE>
```

#### Escaneo de servicios SSL a un Host y guardar el resultado en un archivo XML
```
sslcan <HOST:PORT> --xml=<FILE>
```
