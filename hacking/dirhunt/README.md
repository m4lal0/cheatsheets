# Dirhunt Cheat Sheet

#### Realizar web crawler a un sitio web
```
dirhunt <URL>
```

#### Realizar web crawler e indicar extensiones a buscar
```
dirhunt <URL> -e php,zip,txt
```

#### Realizar web crawler y buscar ficheros interesantes
```
dirhunt <URL> -f access_log,error_log
```
