# Androguard Cheat Sheet

#### Analizar un APK
```
androguard analyze <APP-NAME.apk>
```

#### Crear un archivo de grafico sobre los nodos de conexión del APK
```
androguard cg <APP-NAME.apk>
```

#### Mostrar la versión del APK
```
androguard apkid <APP-NAME.apk>
```

#### Mostrar las versiones de las firmas del APK
```
androguard sign <APP-NAME.apk>
```
