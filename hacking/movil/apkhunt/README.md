# APKHunt Cheat Sheet

#### Escaneo de un APK
```
go run apkhunt.go -p <NAME>.apk
```

#### Escaneo de un APK y guardar los registros
```
go run apkhunt.go -p <NAME>.apk -l
```

#### Escaneo de varios APK's que estan dentro de un directorio
```
go run apkhunt.go -m <NAME-DIRECTORY>
```
