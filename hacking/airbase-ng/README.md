# Airbase-ng Cheat Sheet

#### Crear un AP OPN
```
airbase-ng -c <CHANNEL> --essid <ESSID-AP> -P <INTERFACE-MON>
```

#### Crear un AP WEP
```
airbase-ng -c <CHANNEL> --essid <ESSID-AP> -W 1 -N <INTERFACE-MON>
```

#### Crear un AP WAP
```
airbase-ng -c <CHANNEL> -a <BSSID-AP> --essid <ESSID-AP> <INTERFACE-MON>
```
